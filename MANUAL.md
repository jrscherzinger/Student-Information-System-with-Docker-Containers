# User Manual

This document explains how to use the Student Information System
via the command-line interface.

---

## Usage

docker exec -it sis-app bash
cd /app
./build/sis

Main Menu

1) Add Student
2) List Students
3) Update Student
4) Delete Student
0) Exit

Add Student

Select option 1

Enter:
Name
Surname
Department
Email

List Students

Select option 2

All students stored in the database will be displayed.

Update Student

Select option 3

Enter:
Student ID
New name
New surname
New department
New email

Delete Student

Select option 4

Enter the student ID to delete the record.

Exit Application

Select option 0 to exit the application safely.
